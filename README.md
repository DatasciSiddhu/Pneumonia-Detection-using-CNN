# Pneumonia-Detection-using-CNN


##  Project Overview
Pneumonia is a serious lung infection that can be life-threatening if not detected early. This project uses **Deep Learning (Convolutional Neural Networks - CNN)** to automatically detect pneumonia from chest X-ray images. The model classifies X-ray images into two categories:
- Normal
- Pneumonia

This system helps in **early detection**, **reducing diagnosis time**, and **assisting doctors** in medical decision-making.

---

##  Problem Statement
Manual diagnosis of pneumonia from chest X-rays is time-consuming and depends heavily on radiologist expertise. In many rural or under-resourced hospitals, radiologists may not be available. Therefore, an automated system is needed to detect pneumonia quickly and accurately using deep learning techniques.

---

##  Solution
We built a **CNN (Convolutional Neural Network)** model that:
- Takes chest X-ray images as input
- Performs image preprocessing
- Extracts features using convolution layers
- Classifies the image as Pneumonia or Normal
